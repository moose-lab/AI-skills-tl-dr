# Claude Cowork TL;DR — 让 AI 直接操作你的文件夹

> **适合谁**：想让 Claude 直接操作本地文件的开发者/高级用户
> **前置要求**：macOS + Claude Max 订阅（Pro 用户可加入 waitlist）
> **阅读时间**：3 分钟

---

## 1. 一句话定义

**Claude 可以直接读写你授权的文件夹，自动完成文件整理、内容提取、文档生成等任务。**

> 💡 类比：从「Claude 告诉你怎么做」→「Claude 帮你做完」，像给虚拟同事留便条一样下达任务。

---

## 2. 解决什么问题？

| 传统方式 | Cowork 方式 |
|---------|------------|
| Claude 生成文本 → 你手动复制到文件 | Claude 直接创建/修改文件 |
| 每次任务需手动提供上下文 | Claude 直接读取文件夹内容 |
| 单任务逐个处理 | 多任务并行执行 |

**典型场景**：
- 批量整理下载文件夹（分类、重命名）
- 从多张发票截图提取金额、日期，生成表格
- 将散乱笔记整合成报告初稿

---

## 3. 适用边界

| 适合 | 不适合 |
|-----|-------|
| 批量文件处理（整理、分类、重命名） | 涉及敏感/机密文件（会上传到云端沙箱） |
| 从非结构化内容生成结构化文档 | 需要精确控制的代码开发（用 Claude Code） |
| 重复性办公任务自动化 | Windows 用户（暂不支持） |

---

## 4. 核心机制（3 点）

1. **隔离运行**：文件在 Anthropic 云端沙箱处理，非本地计算
2. **权限边界**：只能访问你明确授权的文件夹，无法越界
3. **确认机制**：删除/重命名等破坏性操作前会请求确认

---

## 5. 快速上手（5 分钟）

### Step 1：启用 Cowork
```
Claude Desktop → 左侧栏 → 点击 "Cowork"
```

### Step 2：授权文件夹
选择一个测试文件夹（建议先用非重要文件夹）

### Step 3：下达任务
```
示例 Prompt：
"分析这个文件夹里的所有 PDF，列出每个文件的主题和页数，
生成一个 summary.md 文件"
```

### Step 4：观察执行
Claude 会：
1. 制定计划并展示
2. 逐步执行，实时汇报进度
3. 关键操作前请求确认

### 预期结果：
- 文件夹中出现 `summary.md`
- 内容包含每个 PDF 的主题和页数表格
- Claude 汇报「任务完成」

### 进阶：配合 Chrome 扩展
```
安装 Claude in Chrome → Cowork 可执行涉及网页的任务
（如：搜索信息后写入本地文件）
```

---

## 6. 常见误区

| ❌ 错误做法 | ✅ 正确做法 |
|------------|------------|
| 「删除旧文件」（模糊） | 「删除 2023 年之前创建的 .tmp 文件」（明确范围） |
| 处理公司机密文件 | 仅处理非敏感数据（文件会上传到沙箱） |
| 直接处理来源不明的文件 | 警惕 Prompt Injection，先检查内容 |
| 多任务同时操作同一文件 | 分步执行，避免竞争条件 |

---

## 7. 当前限制

| 限制项 | 状态 |
|-------|-----|
| 平台 | 仅 macOS |
| 订阅 | Max（Pro 需 waitlist） |
| 跨设备同步 | 暂不支持（规划中） |
| 运行环境 | 云端沙箱（非本地） |
