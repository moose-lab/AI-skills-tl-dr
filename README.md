# AI Skills TL;DR

AI 相关技术的理论和实践正在快速膨胀，但真正有用的信息往往被淹没。

这个仓库的目标是：

> **用 TL;DR 的形式，帮助你在最短时间内学会并使用 AI 技能。**

---

## 这是一个什么项目？

这是一个 **关注AI-skills 的 TL;DR 知识仓库**。

我们关注的是：
- 实际能不能用
- 什么时候得用
- 用错会发生什么

而不是：
- 长篇理论
- 重复解释
- 行业观点或趋势分析

---

## TL;DR 在这里意味着什么？

在本项目中，TL;DR 指的是：

- 去掉无关信息
- 只保留决策与行动所需内容
- 帮助你 **快速理解 → 快速实践**

每一篇 TL;DR 都遵循**统一结构**，用于保证最小质量，而不是追求完美解释。

严格按照TLDR_TEMPLATE.md格式，用AI和实践反馈，双重检验和筛选真正有价值，最短时间学习AI技能的方法论。

---

## 什么是 AI Skill？

在这里，AI Skill 指：

- 可学习、可实践、可拿到结果的能力
- 能通过工具、代码或流程体现
- 面向真实使用场景

例如：
- Vibe Coding 的使用方式
- Prompt 设计
- Agent 工作流
- AI 工具的实际用法
- 用 AI 提升工程或产品效率的方法

不包括：
- 纯观点或趋势判断
- 无法落地的理论内容

---

## 内容来源要求

只接受**真实一手信息**：

### 接受：
- 官方文档 / 技术规范
- GitHub 仓库
- 原始技术说明与示例

### 不接受：
- 二次、三次加工内容
- 内容农场或营销文章
- 无法追溯原始来源的信息
